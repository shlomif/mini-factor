# mini-factor
A limited and simplified implementation of the Unix
[factor](https://en.wikipedia.org/wiki/Factor_(Unix)) program for
[factorisation](https://en.wikipedia.org/wiki/Factorization).

Example session:

```
$ ./mini_factor 100
100: 2 2 5 5
```

# Building:

```
mkdir b
cd b
cmake ..
make
```
